
析构函数不要抛出异常，如果析构函数内抛出异常应该直接处理，或者结束程序。

看这个例子:

```C++
#include<iostream>
#include<vector>
using namespace std;

class A {
private:
    vector<int> vec{1, 2, 3, 4, 5};
public:
    ~A() {
        for (int i : vec) {
            if (i > 3) {
                throw("exception");
            }
        }
    }
};


int main(int argc, char const *argv[])
{
    A a;
    return 0;
}
```

当 `~A()` 执行，大于 3 的时候会抛出一个异常，此时如果不捕获该异常，则程序就会崩溃掉。

如果vec 中是一个堆上的对象，则会发生内存泄漏。

因为有资源并未释放掉。
